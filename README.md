#installs

Vim 
====
sudo apt-get install vim

Haskell Compiler
===============
sudo apt install ghc

Terminal Stuff
==============
bash git prompt
--------------
git clone https://github.com/magicmonty/bash-git-prompt.git ~/.bash-git-prompt --depth=1

Add to bashrc

GIT_PROMPT_ONLY_IN_REPO=1

source ~/.bash-git-prompt/gitprompt.sh
